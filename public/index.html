Note Exchange's purpose

Every student can find themselves looking for study materials. Unsurprisingly, some of them are quite expensive, difficult to find or apparently unavailable. That inspired me to create a personal project that focuses on easily sharing study resources.

Since SQL is still new to me, I figured that the best way to get the hang of it is to use it in a real project - a simple Server-rendered app, made with Express.js, that communicates with a remote PostgreSQL DB.

How to use note Exchange?

Currently, Note Exchange is in an initial state of development - you can use it with Postman or by simply checking out the created endpoints.
